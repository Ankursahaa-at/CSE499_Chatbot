Basic Requirement:
+ Install latest version of Anaconda
+ Form Anaconda Navigator create Python Environment with python 3.7 
Note: If there 3.7 not available then click on Anaconda Prompt and 
type, "conda install python=3.7 anaconda=custom"
+ Intsall latest version of Tensorflow, Keras, Nltk, Flask.
+ Open terminal from created Environment, Install pickle from Anaconda prompt.

install:
conda install -c conda-forge pickle5
conda install -c anaconda colorama
